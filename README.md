# 2bit Yootnori

Logic Gate Powered · 2P Pass & Play

- Framework: React + Vite
- Deploy: GitHub Pages (dist)

## Scripts

- Dev: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`

## GitHub Pages

- `vite.config.js`의 base가 `/2bitYootnori/`로 설정되어 있습니다.
- GitHub Pages(Source: GitHub Actions) 설정 후 main/master로 push하면 자동 배포됩니다.
